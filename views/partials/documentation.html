Your game picker function is provided three variables: team1, team2 and game.

Each of these variables holds an object with important information to help you determine the winner of the game. You can reference the stat you want by using the "dot notation", like team1.losses or team1.win_pct or game.round.

game Object

Property Description
roundRound of current game. Value ranges from 0 to 5 where 5 is the first round and 0 is the championship.
dateDate of game

Writing the function
  • The game picking function definition should look like this: function (game, team1, team2) {}
  • To indicate a team's win, call the function: team1.winsGame() or team2.winsGame()
  • To give both team1 and team2 a chance to win, make sure both teams have their respective winsGame() function called at some point in your logic.
